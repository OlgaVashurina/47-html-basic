JS DOM

Все элементы на web странице можно представить в виде глобального объекта document и иметь к ним доступ

чтобы находить элементы в DOM можно использовать методы поиска отличительным особенностям элементов:

  • тегам - document.getElementsByTagName()
  • классам - document.getElementsByClassName()
  • id - document.getElementById()
  • document.querySelector() - универсальный метод. Класс ищется через обращение через точку, id - через #

Обратите внимание что значение в круглых скобках для поиска передается в виде строки, т.е в кавычках (двойных, одинарных или косых)
